# SMB Monitor

Общий дэшборд платформы мониторинга ключевых показателей бизнеса.
Текущая версия использует агрегированные тестовые данные из Excel-таблицы распределения денежных средств за 16.06.2026.

## Стек

- Vite
- React
- TypeScript
- CSS

## Локальный запуск

```bash
npm install
npm run dev
```

После запуска открой локальную ссылку, которую покажет Vite. Обычно это:

```text
http://127.0.0.1:5173/
```

## Проверки

```bash
npm run typecheck
npm run build
```
