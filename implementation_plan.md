# План реализации SMB Monitor

## Текущий выбранный стек

- Frontend: Vite, React, TypeScript.
- Стили: обычный CSS с проектными переменными, без UI-фреймворка на старте.
- Сборка и локальная разработка: npm scripts `dev`, `typecheck`, `build`, `preview`.
- Причина выбора: платформа сейчас начинается как интерактивный кабинет с дэшбордами, вкладками, формами и состояниями доступа; React дает компонентную структуру, TypeScript снижает риск ошибок в ролях и данных, Vite быстро запускает локальную разработку без преждевременного серверного слоя.
- На следующих этапах backend, авторизация, база данных, интеграции и уведомления выбираются отдельно после уточнения требований к размещению, безопасности и источникам данных.
- Текущая стартовая страница использует тестовую сводку из Excel-файла `Распределение за 16.06.2026г..xlsx` и отображает единый итог аналитики данных: расчетный статус дня, положительный денежный поток, покрытие платежей, долю продаж, donut-структуру расходов, waterfall движения денег, автоматически сформированные выводы, действия и проверочную детализацию.
